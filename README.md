# 🚀 Gestion de Projets IA

Une application complète de gestion de projets avec intelligence artificielle, développée avec Streamlit et déployée sur Hugging Face Spaces.

## 🎯 Aperçu

Cette application offre une solution complète pour la gestion de projets avec des fonctionnalités avancées de visualisation et d'analyse. Elle comprend plusieurs vues spécialisées pour différents aspects de la gestion de projet.

## ✨ Fonctionnalités

### 🏠 Tableau de Bord
- **Métriques en temps réel** : Total projets, projets actifs, taux de completion
- **Visualisations interactives** : Graphiques de répartition par statut et priorité
- **Projets récents** : Vue d'ensemble des derniers projets créés
- **Export des données** : Possibilité d'exporter en Excel

### 📋 Gestion des Projets
- **Liste complète** avec filtres avancés (statut, priorité, recherche)
- **Détails enrichis** : Informations complètes sur chaque projet
- **Actions rapides** : Visualisation, modification, suppression

### 📊 Nomenclature (BOM - Bill of Materials)
- **Gestion des matériaux** : Liste complète des composants
- **Calculs automatiques** : Coûts totaux et moyens
- **Analyses visuelles** : Graphiques de répartition des coûts
- **Fournisseurs** : Suivi des sources d'approvisionnement

### 🛠️ Itinéraire de Fabrication
- **Opérations détaillées** : Séquences de production
- **Temps et ressources** : Estimation des durées et allocation
- **Suivi des statuts** : Progression en temps réel
- **Analyses** : Répartition par statut et ressource

### 📈 Vue Gantt
- **Planning visuel** : Diagramme de Gantt interactif
- **Chronologie** : Visualisation des échéances
- **Codes couleur** : Différenciation par statut
- **Statistiques** : Durées moyennes, projets les plus longs/courts

### 📅 Vue Calendrier
- **Navigation mensuelle** : Exploration par mois
- **Événements** : Dates de début et fin des projets
- **Codes couleur** : Différenciation des types d'événements
- **Détails contextuels** : Informations sur chaque événement

### 🔄 Vue Kanban
- **Colonnes par statut** : À FAIRE, EN COURS, EN ATTENTE, TERMINÉ
- **Cartes projet** : Informations essentielles sur chaque projet
- **Codes couleur** : Priorités et statuts visuels
- **Organisation visuelle** : Vue d'ensemble de l'avancement

## 🔧 Installation et Utilisation

### Prérequis
- Python 3.8+
- Streamlit
- Pandas
- Plotly

### Installation Locale
```bash
git clone https://huggingface.co/spaces/[VOTRE_USERNAME]/gestion-projets-ia
cd gestion-projets-ia
pip install -r requirements.txt
streamlit run app.py
```

### Utilisation sur Hugging Face
Accédez directement à l'application déployée sur Hugging Face Spaces. Aucune installation requise !
